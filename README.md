# AOI 검증 프로그램

> 한국어 사용자 인터페이스(GUI)를 제공하는 AOI(자동 광학 검사) 검증 도구입니다.

- **개발자:** 임현택 (HyunTaek Lim)
- **소속:** Bump 2 Dept. / AOI Engineer
- **실행 환경:** Spyder IDE 내부에서 실행 (별도 exe 빌드는 계획에 없음, 단 PyInstaller 호환은 유지)

---

## 1. 설치

```bash
pip install -r requirements.txt
```

> `torch`, `torchvision` 은 CNN 임베딩을 활성화할 때만 추가로 설치하세요.

## 2. 폰트 안내

한국어가 깨지지 않도록 다음 폰트 중 하나가 시스템에 설치되어 있어야 합니다.

- `Pretendard`
- `Noto Sans KR`
- `Malgun Gothic` (Windows 기본)

영문 타이틀은 `Orbitron`, 본문은 `Rajdhani` 를 우선 사용하며,
한글이 포함되면 한국어 폰트로 자동 폴백됩니다.

## 3. Spyder 에서 실행하기

Spyder 의 IPython 콘솔은 자체 Qt 이벤트 루프를 운영하므로 그대로 실행하면 GUI 가
멈추거나 두 번째 실행이 되지 않습니다. 다음 절차를 따라 주세요.

1. Spyder → **도구(Tools) > 환경설정(Preferences) > IPython 콘솔 > 그래픽** →
   **Backend = `Qt5` 또는 `Qt`** 로 설정합니다.
2. 또는, 콘솔에서 한 번만 `%gui qt` 를 입력한 뒤
   `%runfile main.py` 로 실행합니다.
3. **실행 모드는 “전용 콘솔에서 실행(Execute in a dedicated console)” 을 권장**
   합니다. 매 실행마다 깨끗한 환경을 보장합니다.

코드의 진입점인 `main.py` 는 이미 존재하는 `QApplication.instance()` 를 재사용
하도록 작성되어 있으므로 Spyder 내부에서 안전하게 동작합니다.

## 4. 폴더 구조 가정

```
[기준 장비 최상위 폴더]
├── Slot_01/
│   ├── image1.jpeg
│   └── ...
├── Slot_02/
└── ...

[검증 장비 최상위 폴더]   ← 기준 장비와 동일한 Slot 명을 사용해야 매칭됩니다
├── Slot_01/
├── Slot_02/
└── ...
```

- 두 최상위 폴더는 **서로 다른 호기(장비)** 의 결과물입니다.
- 같은 경로를 두 번 입력하면 경고 모달이 표시됩니다.
- 한쪽에만 존재하는 Slot 은 결과 보고서의 `Slot 불일치 목록` 에 기록됩니다.

## 5. 사용 흐름 (요약)

1. 모드 선택 (한쪽만 검증 / 양쪽 교차검증)
2. 기준·검증 장비 폴더와 호기 번호 입력, 유사도 임계치 설정 후 **`검증 시작`**
3. **Stage 1** — 후보 선별 (중앙 사진에 대해 `검증` 또는 `제외` 결정)
4. **Stage 2** — 유사도 기반 매칭 (기준 사진 1장에 대해 유사한 검증 사진 선택)
5. 결과를 `양식.xlsx` 템플릿에 채워서 저장

## 6. 키보드 단축키

| 단축키 | 동작 |
| --- | --- |
| `←` 또는 `1` | 검증 |
| `→` 또는 `2` | 제외 |
| `Z` | 직전 결정 취소 (Undo) |
| `S` | Stage 2 에서 Skip |

## 7. 캐시 위치

썸네일·중간 크기 이미지·세션 상태는 다음 경로에 저장됩니다.

```
~/.aoi_verification_cache/
```

원본 폴더를 절대로 수정하지 않습니다.

## 8. 결과 파일

- 파일 이름: `AOI검증결과_{기준호기}_vs_{검증호기}_{YYYYMMDD_HHMMSS}.xlsx`
- 동일 폴더 안에 `양식.xlsx` 를 두면 템플릿으로 자동 인식합니다.
- B 열: Slot 명 / C 열: 기준 장비 사진 / D 열: 검증 장비 사진
- 교차 검증 모드에서는 E 열 (매칭 방향) 과 미탐 시트가 추가로 생성됩니다.
